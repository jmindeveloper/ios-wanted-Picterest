# 프로젝트 소개
### Picterest
- 서버에서 이미지를 받아와 가변레이아웃으로 나타냅니다.
- 원하는 사진을 저장 혹은 삭제할 수 있습니다.

<br/>

# 개발 기간
- 2022.07.25 ~ 2022.07.28

<br>

# 사용 기술
- `UIKit`, `MVVM`, `FileManager`, `CoreData`, `NSCache`, `Combine`, `URLSession`, `UICollectionViewLayout`, `NoStoryboard`
